# Diet Plan & BMI Calculator

## Overview

This is a Python-based console application that calculates the user's Body Mass Index (BMI) and suggests diet plans based on BMI category and dietary preference. The project is designed for learning Python fundamentals, conditional statements, dictionaries, input validation, Pandas, and the Datetime module.

## Features

- User login authentication
- Displays current date and day
- Calculates Body Mass Index (BMI)
- Classifies BMI into:
  - Underweight
  - Normal Weight
  - Overweight
  - Obesity
- Suggests weight gain or weight loss plans
- Supports three diet preferences:
  - Vegetarian
  - Vegan
  - Non-Vegetarian
- Input validation for user choices
- Displays BMI category table using Pandas DataFrame
- Educational disclaimer before displaying diet recommendations

## Technologies Used

- Python
- Pandas
- Datetime Module

## Program Flow

1. User Login
2. Enter Age, Weight and Height
3. BMI Calculation
4. BMI Category Display
5. Select Goal (Weight Loss or Gain)
6. Select Diet Preference
7. View Diet Plan Recommendation

## Disclaimer

This project is developed for educational purposes only. The diet plans provided are sample recommendations collected from publicly available sources and should not be considered medical or nutritional advice. Always consult a qualified healthcare professional or registered dietitian before following any diet plan.
